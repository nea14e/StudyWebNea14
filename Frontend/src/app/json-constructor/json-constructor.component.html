JSON - это формат данных, который позволяет отобразить все обычные программистские типы данных в виде текста. По протоколу HTTP REST можно передавать только текст. Поэтому, чтобы передать любые данные, нужно сначала превратить их в текст, и JSON - удобный способ это сделать.

Возможные значения JSON - это

  • булево значение true/false;
  • число 123, -123, 1.023;
  • строка "Qwerty123";
  • null;
  • объект;
  • список.

В свою очередь, объекты и списки состоят из вложенных в них значений JSON, то есть, в объект могут быть вложены другие объекты, списки и тому подобное.
